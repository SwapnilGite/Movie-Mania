## API Reference

#### Get movies by genre

```http
  GET /genre/list
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `Genre` | `string` | api_key=adfe28716db7626ea54e19adc5569c0|


#### Get trending movies by week

```http
  GET /trending/Movie/week
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `time_window`      | `string` |Required.api_key  |
